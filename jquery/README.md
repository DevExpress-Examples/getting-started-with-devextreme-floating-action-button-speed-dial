# DevExtreme jQuery Floating Action Button Multiple Actions - Getting Started 

This repository stores the code examples of the jQuery Floating Action Button component for the [Getting Started with Floating Action Button - Multiple Actions](https://js.devexpress.com/Documentation/Guide/UI_Components/Floating_Action_Button/Getting_Started_with_Floating_Action_Button/#Multiple_Actions_Speed_Dial) tutorial. The FAB allows users open a menu with several related actions (speed dial).

To launch the project, open `index.html` with your browser.
